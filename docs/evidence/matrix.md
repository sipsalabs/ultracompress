# UltraCompress evidence matrix

This evidence summary has been moved to NDA-only access during a documentation refresh.

For the canonical public registry of UltraCompress verified architectures — **22 PPL-verified architectures** (17 dense + 4 MoE + 1 SSM, the SSM with an explicit comparator-note caveat per the registry) **plus 1 ViT cosine-verified** (DINOv2-Large; cosine metric documented separately) — across 4 architecture classes (0.6B–405B) — see:

- [README on GitHub](https://github.com/sipsalabs/ultracompress#readme) — top-line numbers, install, quickstart
- [HF organization](https://huggingface.co/SipsaLabs) — public model packs (PPL-verified subset published; additional packs gated:manual under MOAT policy)
- [docs/benchmarks.json](../benchmarks.json) — registry source-of-truth
- [docs/concepts/catastrophic-failures.md](../concepts/catastrophic-failures.md) — refuted-with-mechanism catalog

Detailed per-model evidence (per-tensor SHA-256 manifests, full eval-pipeline logs, internal benchmark archives) is available to partners under NDA — contact **legal@sipsalabs.com**.
